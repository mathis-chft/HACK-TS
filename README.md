# Installation
## 1. Clone the repo

```bash
 git clone https://github.com/mathis-chft/HACK-TS.git
```
## 2. install the dependencies
```bash
 cd HACKTS
```
```bash
 npm i
```
## 3. Launch the backend
```bash
 uvicorn backend:app --reload
```
## 4. Launch the server
```bash
 cd HACKTS/cv-optimizer
```

```bash
 npm run start
```
